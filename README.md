# AWS Three-Tier Web Application

## 🚀 Overview
This project demonstrates deployment of a scalable three-tier web application on AWS, consisting of frontend, backend, and database layers.

## 🧠 Architecture
The system follows a three-tier architecture:

- **Presentation Tier:** Frontend hosted on EC2 (Apache)
- **Application Tier:** Backend server (Node.js) on EC2
- **Data Tier:** MySQL database hosted on Amazon RDS

It also includes:
- Custom VPC
- Public & Private Subnets
- Internet Gateway & Route Tables
- Application Load Balancer

## ⚙️ Key Features
- Scalable cloud architecture using AWS services
- Secure networking using VPC and subnets
- Load balancing for traffic distribution
- Backend communication with database

## 🛠️ Technologies Used
- AWS (EC2, RDS, S3, Load Balancer)
- Node.js
- MySQL
- Apache Server


## 📚 What I Learned

- Networking concepts (VPC, Subnets, Routing)
- Deploying scalable applications
- Load balancing and high availability
